A color management app for desktop. Easily manage your color palettes for graphic design, art, and more.

![Home Screen](screenshots/Screenshot%202026-01-21%20225911.png)

## Features

- **Create multiple color palettes**
- **Multiple methods for selecting colors**
- **No pesky save button**
- **Responsive Design**

## Installation

### Windows installer
1. Download the latest installer
2. Run it and follow the on-screen instructions.

### From Source
```bash
# Clone the repo
git clone

# Install dependencies
flutter pub get

# Run the app
flutter  run -d windows
```
